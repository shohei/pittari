Pittari
===

# What's this?
Pittari controller for OpenSCAD
WIP: Currently only viewer implemented

# Install
```sh
$ git clone 
```

# How to use
### Initialize
```sh
$ bower install
$ npm install
```
### Run
```sh
$ ./serve.sh <YOUR SCAD FILE>
$ ./scadview.sh <YOUR SCAD FILE>
``
You will see the view updated while editing scad file (auto-update)
